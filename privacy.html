De Inloop respecteert uw privacy. Deze pagina legt uit hoe we omgaan met informatie op onze website.

Welke gegevens verzamelen we?

We verzamelen geen persoonlijke gegevens via deze website. Er is geen registratie, geen login voor bezoekers en geen contactformulier.

Hosting

Deze website wordt gehost via GitHub Pages. GitHub kan technische loggegevens (zoals IP-adressen) bijhouden conform hun eigen privacybeleid: github.com.

Cookies
